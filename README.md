# CameraApp

In order to run and check this simple app, you need to follow some steps.

1. Clone the project
2. Run "npm install" - Only for iOS "cd ios && pod install && cd .." 
3. Build "npm run ios/android"
4. Start the server "npm start"
5. Done!
